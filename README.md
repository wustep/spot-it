# Spot it!

A web-based implementation of the Spot it! / Dobble card game, built with Next.js.

[Buy the game](https://amzn.to/3Kxu06V).


## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to play.
